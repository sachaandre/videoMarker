Marked Videos - Demo

Informations on options :

  • "Return on pause" : Go back to the starting point of the last marker you passed
  • "Pause at End of Mark" : Automaticly stop a video when reaching the end of a marked section

How to use MarkedVids ?

A MarkedVid is a class which helps you to create videos elements and append markers to them.

How to create a marked vid ?
let myInstance = new MarkerVid("path of your video", "video/type");
By default, the end of a marker is total length of the video
